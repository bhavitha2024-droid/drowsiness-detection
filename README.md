# Drowsiness Detection System

This project detects driver drowsiness using computer vision.

## Features
- Real-time face detection
- Eye tracking
- Alert system when drowsiness is detected

## Technologies Used
- Python
- OpenCV
- Machine Learning

## How to Run
1. Install requirements:
   pip install -r requirements.txt

2. Run:
   python app.py
